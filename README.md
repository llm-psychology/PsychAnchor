# PsychAnchor
A dynamic personality calibration system for LLM agents that quantifies psychological traits and maintains behavioral consistency through real-time anchoring algorithms. Insipred by PersonaGym.

LLM分數產生的三大缺陷：
無理論錨定：缺乏心理測量學基礎
低復現性：相同輸入產生波動性輸出
邏輯斷裂：各維度分數缺乏內在一致性
